Exploração de dados sobre a história recente dos jogos olímpicos. Neste projeto explorei um conjunto de dados sobre os Jogos Olímpicos modernos, incluindo todos os jogos de Atenas (1896) ao Rio (2016). Foram realizadas para as análises a linguagem Python e suas respectivas bibliotecas (numpy/pandas/seaborn/matplotlib).

Neste projeto explorei um conjunto de dados sobre a série de grande sucesso - Game of Thrones -. A série é composta por 8 temporadas e 73 episódios no total. Para a análise dos dados foi utilizado o Python e algumas de suas bibliotecas para apoio.

Painel financeiro, idealizado com dados fictícios, com o objetivo de apresentar visualmente, informações macros referentes ao negócio. Tal como é viável filtrar os dados por ciclos temporais e instituições bancárias envolvidas nas transações.

Dashboard desenvolvido para a área de R.H., baseado em dados fictícios, com a incubência de projetar visualmente, elementos relevantes do atual cenário do setor. Explorou-se para os visuais, sparkline, gráfico de árvore hierárquica, gráfico funil e gráfico rosca.

Dashboard de vendas criado com o objetivo de fornecer aos gestores das áreas comerciais e/ou administrativas dados que lhes auxiliem em sua tomada de decisões. É um relatório composto pelo faturamento, departamentos, produtos, vendedores e evolução de vendas por períodos.

Dashboard desenvolvido para a área de R.H. O propósito deste relatório é fornecer informações essências sobre o panorama dos colaboradores. Este relatório é constituído de indicadores como quantidade de colaboradores ativos, desligados e admitidos, com filtros por gênero, função e modalidade de trabalho.

Este Dashboard apresenta relevantes informações para uma empresa fictícia do ramo de Pet Food, apontando o faturamento total por período, faturamento por gênero, quantidade de produtos vendidos, média de pets por clientes, entre outros.

Interface criada para um sistema de gerenciamento de registro ponto. O desenvolvimento do projeto teve como premissas a usuabilidade dos usuários e os requisitos do cliente. O produto será de uso tanto de gestores como de colaboradores. Este exemplo contempla as rotas para os gestores.

Este dashboard foi desenvolvido para auxiliar a área de produção/ qualidade de uma organização fictícia. São KPI's que apontam disponibilidade, índice de qualidade, quantia produzida e rejeitada de peças, horas paradas, entre outros, podendo serem filtrados por operadores e meses.

Dashboard desenvolvido em excel, voltado para a área de logistica. Painel composto por KPI'S imprescindíveis para companhias de transporte que visam medir a qualidade geral em todas as áreas do processo de entrega.

Script criado para atender a proposta de desenvolvimento do sistema de gerenciamento de registro ponto. Contem-pla cadastros e consultas para a geração de relatórios.
